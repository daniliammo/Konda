# Публикация сайта Konda через GitHub Pages

Готовый сайт находится в отдельной ветке `website`, в папке `docs`.

## 1. Скачать ветку сайта

На Linux открыть терминал в папке проекта Konda и выполнить:

```bash
git fetch origin
git switch website
git pull
```

## 2. Посмотреть сайт локально

```bash
cd docs
python3 -m http.server 8000
```

Открыть в браузере:

```text
http://localhost:8000
```

Для остановки сервера нажать `Ctrl+C`.

## 3. Объединить сайт с основной веткой

Если сайт устраивает:

```bash
cd ..
git switch main
git pull origin main
git merge website
git push origin main
```

Перед объединением проверить, что новые изменения проекта не конфликтуют
с файлами в папке `docs`.

## 4. Включить GitHub Pages

1. Открыть репозиторий `daniliammo/Konda` на GitHub.
2. Перейти в `Settings`.
3. Открыть раздел `Pages`.
4. В разделе `Build and deployment` выбрать `Deploy from a branch`.
5. Выбрать ветку `main`.
6. Выбрать папку `/docs`.
7. Нажать `Save`.

Через несколько минут сайт должен появиться по адресу:

<https://daniliammo.github.io/Konda/>

## 5. Что проверить

- главную страницу на компьютере и телефоне;
- примеры кода Konda;
- актуальность списка возможностей и дорожной карты;
- ссылку на GitHub;
- ссылку на презентацию.

Основные файлы сайта:

```text
docs/index.html
docs/style.css
docs/script.js
docs/favicon.svg
docs/.nojekyll
```

Остальные презентации и PDF из папки `docs` удалять не нужно.
