01 · Язык

Знакомые идеи.
Понятные слова.

Konda сохраняет строгость системного программирования, но делает исходный код ближе к тому, как мы объясняем алгоритмы друг другу.

01

Без лишней пунктуации

Перевод строки завершает команду. Точка с запятой нужна только там, где действительно помогает структуре.

целое32 ответ = 42
ответ = ответ + 1
printf("%d\n", ответ)
↻

Условия и циклы

если, иначе, пока и для уже работают.

8→64

Точные типы

целое8…целое64, байты, символы, вещественные числа и указатели.

⌁

Проверки до запуска

Необъявленные переменные, неверная область видимости и деление на ноль становятся понятными ошибками.

→ C

Работает с экосистемой C

Результат — читаемый C-код и нативная программа, собранная GCC или Clang.

Один алгоритм · два представления

Konda говорит понятно. Компьютер получает C.

Это не онлайн-компилятор, а наглядный пример того, что делает локальный транспилятор.

счётчик.конда 
целое32 точка_входа(
    целое32 количество,
    символ** аргументы)
{
    целое32 счётчик = 3

    пока (счётчик > 0) {
        printf("%d\n", счётчик)
        счётчик = счётчик - 1
    }

    вернуть 0
}
#include <stdint.h>
#include <stdio.h>

int32_t main(
    int32_t argc,
    char **argv)
{
    int32_t счётчик = 3;

    while (счётчик > 0) {
        printf("%d\n", счётчик);
        счётчик = счётчик - 1;
    }

    return 0;
}
